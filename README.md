# 💻 Laptop Price Prediction | Machine Learning Project

## 📌 Project Overview

This project focuses on building a machine learning model to accurately predict laptop prices based on various specifications. The solution is designed for SmartTech Co. to support competitive pricing strategies and better market positioning.

With the growing variety of laptops in the market, predicting prices based on features like brand, RAM, processor, and storage is crucial for both consumers and businesses.

---

## 🎯 Objectives

* Develop an accurate laptop price prediction model
* Identify key factors influencing laptop pricing
* Analyze the impact of brand on pricing
* Enable real-time predictions for new laptop entries

---

## 📊 Dataset Description

The dataset includes various laptop specifications such as:

* Brand
* Processor
* RAM
* Storage (HDD/SSD)
* GPU
* Screen Size
* Operating System
* Weight
